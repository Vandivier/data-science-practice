step 1: Install R from CRAN
step 2: Add R CLI tool (Rcmd.exe) to PATH (default location C:\Program Files\R\R-3.2.3\bin\x64)
step 3: Open CLI and type "r" or "R" to start an R session on CLI.
step 4: Exit/Quit/Terminate session with CLI command "quit()" or alias "q()"
step 5: get these other things:
        Packrat   - An R package manager
        Rscript   - A library which allows you to pipe R values to the console or standard out.
                    Instead of the default behavior of R which is to create a new file.
        Shiny     - A web app framework for R.
        Rstudio   - An IDE for R.
